# Учебный компилятор / интерпретатор

Собственный язык программирования и древовидная виртуальная машина на Go.
Реализованы все 7 лабораторных работ в виде единого пайплайна компилятора.

## Что сделано

| ЛР | Этап | Пакет | Реализовано |
|----|------|-------|-------------|
| 1 | Лексический анализ | `lexer` | Токенизация (Maximal Munch): ключевые слова, числа, строки с escape, комментарии `//` и `/* */` |
| 2 | Синтаксический анализ | `parser` | Рекурсивный спуск, приоритеты и ассоциативность операторов, построение AST |
| 3 | Семантический анализ | `semantic` | Таблицы символов, проверка типов, контроль объявлений, предупреждения о неиспользуемых переменных |
| 4 | Интерпретатор | `interpreter` | Tree-Walk выполнение, динамическая типизация, `if`/`while`, `print`, ленивые `&&`/`\|\|` |
| 5 | Функции и стек вызовов | `interpreter` | Пользовательские функции, аргументы, `return`, рекурсия, изолированные стековые фреймы |
| 6 | Оптимизация AST | `optimizer` | Свёртка констант (Constant Folding), удаление мёртвого кода (Dead Code Elimination) |
| 7 | Массивы | `parser` + `interpreter` | Литералы `[...]`, индексация чтения/записи, `len`, `append` |

Пайплайн: `lexer → parser → semantic → optimizer → interpreter`.

Каждый этап покрыт unit-тестами (`go test ./...`).

## Запуск

```bash
go run . examples/factorial.lang      # выполнить программу
echo 'print 2 + 2;' | go run .        # из stdin
go run . --tokens examples/demo.lang  # вывод потока токенов (ЛР1)
go test ./...                         # запуск тестов
```

## Примеры (каталог `examples/`)

- `factorial.lang` — рекурсия (ЛР5)
- `fibonacci.lang` — рекурсия и циклы (ЛР4/5)
- `bubble_sort.lang` — пузырьковая сортировка массива (ЛР7)
- `demo.lang` — обзор всех возможностей языка

## Пример программы

```c
func factorial(n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}
print factorial(5);

var a = [3, 1, 2];
a[0] = 10;
print a;
print len(a);
```
